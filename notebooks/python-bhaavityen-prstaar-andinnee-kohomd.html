Python භාවිතයෙන් ප්‍රස්තාර අඳින්නේ කොහොමද?

September 01, 2017

Python යනු දත්ත වේදයේ (Data Science) බහුලවම භාවිත වන ක්‍රමලේඛ භාෂාවයි. දත්ත විශ්ලේෂණය පහසු කරවන එක මඟකි, ප්‍රස්තාර. Python යොදා ගෙන ප්‍රස්තාර නිර්මාණයට උපයෝගී කර ගන්නා libraries කිහිපයක්:

  • Matplotlib
  • Seaborn
  • Bokeh

මේවා අතරින් ප්‍රමුඛතමයා matplotlib වේ. seaborn යනු matplotlib හිම දිගුවකි. matplotlib තවම සිංහල බසට පහසුකම් සලසන්නේ නැති බැවින් මේ ලිපියෙන් අවධානය යොමු කරන්නේ bokeh වෙතයි.

bokeh ස්ථාපනය කිරීම¶

Python Package Manager ඔබේ පරිගණකයේ ස්ථාපනය ව පවතී නම්, විධාන පිවිසුමේ (command prompt) හෝ පර්යන්තයේ (Terminal),

pip install bokeh

විධානය මඟින් bokeh පහසුවෙන් ස්ථාපනය කළ හැකිය.

මූලික නිදසුනක්¶

මෙම සරල නිදසුනෙන් මෙරට 2015 වසරේ මාසික වර්ෂාපතනය විචලනය වූ අන්දම නිරූපණය කෙරේ. (මූලාශ්‍රය)
