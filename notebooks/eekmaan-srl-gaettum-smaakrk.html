ඒකමාන සරල ගැටුම් සමාකරකය¶

වස්තුවක් තලයක් මත ගැටුමෙන් පසු පෘෂ්ඨයෙන් ඉවතට චලනය වීම පොලා පැනීමයි. ගැටුමකින් පසු පොලා පැනීමක් සිදු වේ නම් එය ප්‍රත්‍යස්ථ ගැටුමකි. සර්වසම රබර් බෝලයක් හා වීදුරු බෝලයක් මෙලෙස පොලා පනින අවස්ථාවක් සලකමු. සමාන උසක සිට ඒවා අත හැරිය විට ගැටුමට මොහොතකට පෙර 2හිම ප්‍රවේග සමාන වේ. එ නමුත් ගෝල දෙක ඉහළ නගිනුයේ වෙනස් උසවල්ය. ඒ බෝලවල ගැටුමට පසු ප්‍රවේග වෙනස් නිසාය. පොලා පැනීම හා ප්‍රවේග වස්තු සෑදී ඇති ද්‍රව්‍ය මත මෙන්ම ගැටෙන පෘෂඨයේ ස්වභාව මතද රඳා පවතියි. ප්‍රත්‍යස්ථ වස්තු 2ක් ගැටෙන විට ස්පර්ෂ තලයට ලම්බකව ස්පර්ෂ ලක්ෂ්‍යය ඔස්සේ පිහිටි රේඛාව ගැටුම් රේඛාව ලෙස හැඳින්වේ. ගැටුමට බඳුන් වන වස්තු 2හි ස්කත්ධ කේන්ද්‍ර මෙ⁣ම රේඛාව ඔස්සේ පිහිටියේ යයි ගනිමු. ගැටුමට මොහොතකට පෙර අංශු 2හි ප්‍රවේග ගැටුම් රේඛාව ඔස්සේ පවතියි නම්, එම ගැටුම සරල ගැටුමක් ලෙස අර්ථ ගැන්වේ.

රේඛීය ගම්‍යතා සංස්ථිති මූලධර්මයෙන්,

$m_1 v_1 + m_2 v_2 = m_1 u_1 + m_2 u_2\tag{1}$

නිව්ටන් ප්‍රත්‍යාගති මූලධර්මයෙන්,

$u_2 - u_1 = e(v_1 - v_2)\tag{2}$

$(2)\times m_1$ න්,

$m_1 u_2 - m_1 u_1 = e m_1 (v_1 - v_2)\tag{3}$

$(1) + (3)$ න්,

$(m_1 + m_2) u_2 = e m_1 (v_1 - v_2) + m_1 v_1 + m_2 v_2\tag{4}$

එනයින්,

$$u_2 = \frac{(e+1)m_1 v_1 + (m_2 - e m_1) v_2}{m_1 + m_2}\tag{5}$$

$(5)$ හා $(2)$ න්, $u_1$ සෙවිය හැකිය.

collision ශ්‍රිතය පදනම්ව ඇත්තේ ඉහත කරුණු මතයි.

class element(object):
    """වස්තුවක මූලික සැකිල්ල"""
    def __init__(self, m, v, x, y, c):
        self.m = m
        self.v = v
        self.x = x
        self.y = y
        self.c = c
    
    def nextpos(self, dt):
        self.x += self.v * dt
    
    def translate(self, dx):
        self.x += dx

        
class ball(element):
    """අංශුව"""
    def __init__(self, r, m, v, x, y, c):
        self.r = r
        self.m = m
        self.v = v
        self.x = x
        self.y = y
        self.c = c    
    def animate(self):
        return gizeh.circle(self.r, 
                            xy=[self.x, self.y], 
                            fill=(self.c))
    
class wall(element):
    """බිත්තිය"""
    def __init__(self, w, h, x, y, c):
        self.w = w
        self.h = h
        self.x = x
        self.y = y
        self.v = 0
        self.c = c
        self.r = w / 2.
    def animate(self):
        return gizeh.rectangle(self.w, 
                               self.h, 
                               xy=[self.x, self.y], 
                               fill=self.c)

class simulate(object):
    
    def init(self, width, height, elementList, allElements, eList):
        self.t0 = 0
        self.T = 0
        self.width = width
        self.height = height
        self.elementList = elementList
        self.allElements = allElements
            
    def collision(element1, element2, e):
        """ගැටුමට පසු ප්‍රවේග ලබා දේ"""
        v1 = element1.v

        v2 = element2.v

        condition1 = type(element1) is wall
        condition2 = type(element2) is wall

        if condition1 or condition2:
            if condition1:
                return v1, -(e * v2)
            else:
                return -(e * v1), v2

        m1 = element1.m
        m2 = element2.m
        u2 = ((e + 1) * m1 * v1 + (m2 - e * m1) * v2) / float(m1 + m2)
        u1 = u2 - e * (v1 - v2)

        return u1, u2

    def whenCollide(elementList):
        """ඊළඟ ගැටුමට ගතවන කාලය හා සහභාගි වන්නන් ලබා දේ"""
        out = float('inf')
        who = []
        for i in range(len(elementList) - 1):
            element1 = elementList[i]
            element2 = elementList[i + 1]
            rv = element1.v - element2.v
            if rv > 0:
                d = element2.x - element1.x - element1.r - element2.r
                t = d / float(rv)
                if out > t:
                    out = t
                    who = [[i, i+1]]
                elif out == t:
                    who.append([i, i + 1])
        return out, who

    drw = lambda element: element.animate().draw(surface)

    def calNextState():
        """පද්ධතියේ ඊළඟ තත්ත්ව ගණනය කරයි"""
        global time, who, vs, T
        time, who = whenCollide(elementList)
        vs = [0] * len(elementList)
        for pair in who:
            i = pair[0]
            e = es[i]
            element1 = elementList[i]
            element2 = elementList[i + 1]
            vs[i], vs[i + 1] = collision(element1, element2, e)
        T += time

    def updateState():
        """ගැටුමෙන් පසු පද්ධතියේ තත්ත්ව යාවත්කාලීන කරවයි"""
        global t0, T, who
        t0 = T
        for pair in who:
            i = pair[0]
            j = i + 1
            elementList[i].v = vs[i]
            elementList[j].v = vs[j]

    def movElements(t):
        """පද්ධතියේ ගතික සංරචක චලිත කරවයි"""
        global prev_t
        for element in elementList:
            dx = element.v * (t - prev_t)
            element.translate(dx)
        prev_t = t

    def mkf(t):
        """t කාලයක දී පද්ධතියේ ස්වරූපය ලබා දේ"""
        global T, t0, who, vs, prev_t, surface
        surface = gizeh.Surface(width, height)
        map(drw, allElements)
        if t >= T:
            updateState()
            calNextState()
        movElements(t)
        return surface.get_npimage()
